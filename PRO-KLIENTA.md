# Briefing pro nového klienta

Použij tento dokument při prvním rozhovoru s klientem, který chce postavit web na **Krystalické mřížce**. Otázky jsou navrženy tak, aby se nehledalo, jaký by web měl být, ale **jaký už ve své esenci je**.

## 1. Esence

- Kdo jsi, co děláš, pro koho?
- Jaké jméno chceš pro značku?
- Pokud bys měl web popsat **třemi slovy**, jakými?

## 2. Cílová skupina

- Komu je web určen? *(nakresli si konkrétního člověka — věk, situace, hledání)*
- Jaký problém řeší tvůj klient v 3 ráno, když nemůže spát?
- Jaké slova by zadal do Googlu?

## 3. Barvy a tón

- Jaké barvy ti přijdou na mysl, když přemýšlíš o své práci?
- Tichý web (klidný, prostorný) nebo aktivní web (jasné CTA, urgence)?
- Pokud bys měl web přirovnat k prostoru, jaký by to byl? *(katedrála, ateliér, kavárna, lesní mýtina, dílna…)*

## 4. Obsah

### Brána — tři věty otevírající web
Tři krátké věty, které **dýchají**. Postupně se objeví. Otevírají cestu.

Příklad (Morion Light):
> Vše co potřebujete… / Máte v sobě… / Zbývá jen… to znovu objevit.

Tvoje tři věty?

### Síň — sekce hlavní stránky
Standardně: Úvodní vzkaz, Kdo jsi, Co děláš (1–3 služby), Reference, Kontakt, Blog.

Které sekce ti dávají smysl?

### Komnaty — samostatné podstránky
Co potřebuje vlastní hlubší stránku? *(Proč značka, Vzdělání, Reference, Obchodní podmínky, Ochrana osobních údajů…)*

### Vrchol — čtyři věty uzavírající cestu
Zrcadlo k třem větám Brány. Co řekneš návštěvníkovi, který došel až sem?

## 5. Funkce

Zakřížkuj, co web potřebuje:

- [ ] Kontaktní formulář
- [ ] Newsletter signup
- [ ] Online platby
- [ ] Eshop
- [ ] Booking / rezervační systém
- [ ] Blog
- [ ] Analytics (anonymní)

## 6. Technické

- Doména: má klient vlastní? Kde je registrovaná?
- E-mail: má klient vlastní s touto doménou?
- Sociální sítě: které používá?
- Existující web: má ho? Co se z něj má zachovat?

## 7. Časová investice klienta

Krystalická mřížka **vyžaduje, aby klient sám napsal texty**. Šablona dává mřížku, ne příběh. Klient musí:

- Napsat tři věty Brány a čtyři věty Vrcholu
- Napsat sekce Síně (úvodní vzkaz, Kdo jsem, popisy služeb)
- Napsat texty Komnat
- Schválit barevnou paletu

Bez vstupu klienta nelze web personalizovat. Šablona není ChatGPT — drží jen formu, obsah musí přijít zevnitř.
