# Vzorový článek

Tento článek slouží jako vzor pro nové blog příspěvky. Smaž ho, až přidáš první vlastní.

## Jak přidat nový článek

1. Vytvoř nový soubor `nazev-clanku.md` v `src/content/blog/`
2. Vyplň front matter (mezi `---`) — `nadpis`, `popis`, `datum`, případně `autor`
3. Napiš obsah v Markdownu
4. Commit + push → Cloudflare nasadí

## Markdown syntax

**Tučně**, *kurzíva*, [odkaz](https://example.cz).

- Odrážka 1
- Odrážka 2

> Citace

const kod = 'tak';
